## How to Use

1. Open the web UI at `http://localhost:5173`
2. Click "üìÅ Load from File" button
3. Select any example file (JSON or YAML)
4. Form will auto-populate with the example data
5. Click "Generate Campaign" to process

## File Formats

Both JSON and YAML formats are supported:
- `.json` - Structured data format
- `.yaml` - Human-readable format

All examples include:
- Campaign ID
- 2-3 Products with descriptions
- Target region and audience
- Campaign message
- Brand colors
- Copywriting preferences
